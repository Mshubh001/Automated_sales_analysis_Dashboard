# Automated_sales_analysis_Dashboard

Project Overview

This project automates sales reporting and analysis using PostgreSQL and Power BI. The dashboard replaces manual Excel-based reporting by connecting directly to the sales database, providing real-time business insights and automated report distribution.

Key Features

- Direct integration with PostgreSQL database
- Automated data refresh using Power BI Service
- Real-time sales performance monitoring
- Regional sales analysis
- Executive-level KPI dashboard
- Automated report delivery through email subscriptions
- Role-based dashboard sharing and access control
- Elimination of manual Excel-based reporting processes

Dashboard Insights

The dashboard provides:

- Total Sales Analysis
- Quantity Sold Tracking
- Regional Performance Comparison
- Sales Trend Monitoring
- KPI Performance Metrics
- Business Decision Support Reports

Tech Stack

- PostgreSQL
- Power BI Desktop
- Power BI Service
- SQL
- Data Modeling
- DAX
- Data Visualization

Business Impact

- Reduced manual reporting effort
- Improved reporting accuracy
- Automated monthly reporting workflows
- Faster access to business insights
- Enhanced decision-making through real-time analytics

Project Workflow

1. Sales data stored in PostgreSQL database
2. Power BI connected directly to database
3. Data transformation and modeling performed
4. Interactive dashboard developed
5. Scheduled refresh configured in Power BI Service
6. Automated report distribution enabled
7. Leadership receives updated reports automatically
